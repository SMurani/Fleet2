Morris.js Usage

Morris.js is a jQuery based charting plugin created by Olly Smith. In DEF ONE, we are using the most recent version of Morris.js which includes the resize function, which makes the charts fully responsive. The documentation for Morris.js is available on their website, http://www.oesmith.co.uk/morris.js/.
